# 专转本 · 备考日程管理系统

为 SHUO 打造的专转本备考学习追踪系统。

## 功能

- 📊 **仪表盘**：四阶段进度、考试倒计时、统计数据
- 📝 **任务记录**：按科目类型记录学习内容
- 📊 **周报分析**：自动生成每周学习报告
- 📅 **日历视图**：热力图显示学习记录
- ⚙️ **数据管理**：导入/导出/清除数据

## 使用方法

1. 下载所有文件
2. 双击 `index.html` 在浏览器中打开
3. 开始记录你的学习！

## 数据存储

数据存储在浏览器的 LocalStorage 中，记得定期导出备份。

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- 无依赖，纯前端
- LocalStorage 数据持久化
